Provides version information for Maven like repositories.<br/><br/>

<!-- getFileVersion -->

<b><code>getFileVersion</code></b>
<blockquote>
Gets a Maven like file version.<br/><br/>
Maven defines the file version of an artifact that is part of a snapshot build as follows: <code>&lt;baseRevision&gt;-&lt;date&gt;.&lt;time&gt;-&lt;buildNumber&gt;</code>.

<p>
<code>baseRevision</code>
<blockquote>
The base revision of an artifact's version wich is usually the version of the upcoming release.
</blockquote>

<code>release</code>
<blockquote>
Indicates whether to return the file version for a release build or a snapshot build.
</blockquote>

<code>ignoreTaskBranch</code> <i>(optional)</i>
<blockquote>
Disables the usage of the branch name if the current working directory belongs to a task branch.
</blockquote>
</p>

<i>This method returns the branch name if the current working directory belongs to a task branch. Maven like release or snapshot versions are only returned for maintenance and integration branches or if <code>ignoreTaskBranch</code> is set to <code>true</code>.</i>
</blockquote>
